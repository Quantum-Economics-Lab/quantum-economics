# Quantum Economics

This repository contains the foundational theory of Quantum Economics.

The goal is to formally define value as a measurable informational property rather than a purely market-relative quantity.

## Contents
- Definitions and axioms
- Conceptual frameworks
- Working papers
- Formal terminology

This repository is the academic backbone of the Quantum Economics Lab.
